# MCP x402 DePIN Ecosystem

**Tracking DePIN projects adopting or considering the MCP x402 monetization pattern.**

The MCP x402 pattern enables DePIN projects to monetize their data through AI agents and LLMs with per-call pricing, on-chain settlement via Solana, and zero custody.
